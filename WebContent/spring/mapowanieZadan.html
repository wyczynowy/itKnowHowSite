Mapowanie żądań http

Do wskazania kontrolera oraz metody, która ma zostać wywołana do obsługi określonego żądania http służy adnotacja @RequestMapping. Adnotacja ta musi się znaleźć nad klasą lub metodą, która będzie wywoływana w chwili obsługi określonego żądania w klasie kontrolera, czyli klasie oznaczonej adnotacją Springową @Controller:

@Controller
@RequestMapping("/kontroler")
public class AlbumController {

 @Autowired
 private AlbumDao albumDao;

 @RequestMapping("/metoda")
 public String metoda() {
  return "glowny";
 }
}

W powyższym wypadku metoda ta będzie uruchomiona po wejściu na adres http://example.com:8080/aplikacja/kontroler/metoda

@Controller
public class AlbumController {

 @Autowired
 private AlbumDao albumDao;

 @RequestMapping("/metoda")
 public String metoda() {
  return "glowny";
 }
}

W tym wypadku metoda ta będzie uruchomiona po wejściu na adres http://example.com:8080/aplikacja/metoda
