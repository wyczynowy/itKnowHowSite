Instalacja Tomcat

Ubuntu 16.04

1. Pobrać archiwum ze strony http://tomcat.apache.org/download-80.cgi
2. Utworzyć katalog
> sudo mkdir /opt/tomcat
3. Rozpakować poleceniem
> sudo tar xzvf apache-tomcat-8*tar.gz -C /opt/tomcat --strip-components=1
4. Reszta jak tutaj tylko już bez pobierania: https://www.digitalocean.com/community/tutorials/how-to-install-apache-tomcat-8-on-ubuntu-16-04
