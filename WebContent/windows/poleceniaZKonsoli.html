Polecenia z konsoli

Ogólne

> md nowyKatalog - tworzy nowy katalog w bieżącej lokalizacji

> move nazwaPliku nowaNazwaPliku - zmienia nazwę pliku lub katalogu

> move nazwaPliku c:\katalog\innyKatalog - przenosi plik lub katalog w inne miejsce

> rd nazwaKatalogu - usuwa katalog. Parametr /S umożliwia usunięcie katalogu, który nie jest pusty. Parametr /Q umożliwia usunięcie pliku bez konieczności potwierdzania operacji

> del nazwaPliku - usuwa plik

> netstat -o -n -a | findstr 0.0.8080 - wyswietla pid procesu zajmującego port o wskazanym numerze

> taskkill /f /pid 1234 - zabijanie procesu o wskazanym pid

> chcp - Wyświetla aktualne kodowanie znaków ustawione w konsoli

Sieć

> netsh wlan show profile - Wyświetla profile (konfigurację) zapisanych sieci wi-fi

> netsh wlan connect hell - Podłącza się do sieci wi-fi

> netsh wlan disconnect - Rozłącza się z siecią wi-fi

> netsh wlan add profile filename = "C:\OUTPUT\Połączenie sieci bezprzewodowej-hell.xml" - Dodaje profil (konfigurację sieci wi-fi)

> netsh wlan export profile name="hell" folder="C:\OUTPUT\" key=clear - Eksportuje profil (konfigurację sieci wi-fi)

> netsh wlan delete profile "hell"

> netsh interface show interface "Połączenie sieci bezprzewodowej" - Wyświetla stan połączenia "Połączenie sieci bezprzewodowej" Restart wi-fi

> netsh interface set interface name="Połączenie sieci bezprzewodowej" admin=disabled

> netsh interface set interface name="Połączenie sieci bezprzewodowej" admin=enabled

> netsh wlan show networks - Wyświetla dostępne sieci wi-fi
