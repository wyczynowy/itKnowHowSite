Data i czas

Czas i jego formatowanie:

// 10
System.out.println(LocalTime.now().format(DateTimeFormatter.ofPattern("hh")));
// 10:37
System.out.println(LocalTime.now().format(DateTimeFormatter.ofPattern("hh:mm")));
// 10-37-25
System.out.println(LocalTime.now().format(DateTimeFormatter.ofPattern("hh-mm-ss")
));
// 10/37/25/69
System.out.println(LocalTime.now().format(DateTimeFormatter.ofPattern("hh/mm/ss/m
m")));

Data i jej formatowanie:

// 2019-02-08
System.out.println(LocalDate.now());
// 2019-02-08
System.out.println(LocalDate.now().format(DateTimeFormatter.ISO_DATE));
// 20190208
System.out.println(LocalDate.now().format(DateTimeFormatter.BASIC_ISO_DATE));
// 2019-02-08
System.out.println(LocalDate.now().format(DateTimeFormatter.ISO_LOCAL_DATE));
// 2019/02/08
System.out.println(LocalDate.now().format(DateTimeFormatter.ofPattern("yyyy/MM/dd
")));

Data razem z czasem:

// 2019-02-08T23:00:38.209
System.out.println(LocalDateTime.now());
// 2019-02-08
System.out.println(LocalDateTime.now().format(DateTimeFormatter.ISO_DATE));
// 23:00:38.239
System.out.println(LocalDateTime.now().format(DateTimeFormatter.ISO_LOCAL_TIME));
// 2019-02-08 11:00:38
System.out.println(LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyy-M
M-dd hh:mm:ss")));
