Tworzenie projektu maven

Aby móc w Eclipse utworzyć projekt maven dla javy EE, należy najpierw zainstalować maven na komputerze a później doinstalować odpowieni plugin do Eclipse. Drugą opcją jest instalacja odpowiedniej wersji Eclipse wraz z preinstalowanym oprogramowaniem maven. Aby w Eclipse utworzyć projekt maven dla javy EE należy kliknąć File -> New -> Other...

Wybieramy Maven -> Maven Project i przechodzimy do okna ustawień nowego projektu. W tym oknie zaznaczamy opcję Create a simple project (skip archetype selection). Następnie klikamy Next i przechodzimy do kolejnego okna konfiguracyjnego nowego projektu:

W tym oknie ustawiamy Group Id, Artifact Id oraz ustawiamy Packaging wartość war i następnie klikamy przycisk Finish. Projekt zostaje utworzony.

Po wprowadzeniu jakichkolwiek zmian w pliku pom.xml należy kliknąć ppm na nazwę projektu Maven -> Update Project. Tak przygotowany projekt można już skompilować. Kompilację robimy z konsoli poleceniem mvn clean package, przechodząc do folderu projektu, jak pokazano to na załączonym obrazku:
