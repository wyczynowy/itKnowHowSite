Exportowanie programu - możliwość uruchomienia z konsoli systemowej

Aby wyeksportować program napisany w Javie z Eclipse należy kliknąć ppm na projekcie, który chcemy wyeksportować w oknie Project Explorer'a, następnie wybrać opcję Export.... Z listy rozwijanej wybrać Java -> JAR file, następnie klikamy Next. W kolejnym oknie nie trzeba nic zmieniać, więc można przejść do następnego etapu, klikamy znów Next i znów klikamy Next. W ostatnim oknie zatytuowanym JAR Manifest Specification w polu Select the class of the application entry point należy wskazać klasę, będącą punktem wejścia programu, czyli klasę z metodą Main. Klikamy Finish i plik jar programu zostaje utworzony we wskazanej lokalizacji.

Uruchomienie pliku programu .jar

Aby uruchomić tak przygotowany plik jar programu Javy, to należy w konsoli wpisać następujące polecenie:

java -jar nazwaPlikuJarProgramu.jar
