Javascript

Skrypty języka javascript można umieszczać na stronie HTML na kilka sposobów. Pierwszy z nich to umieszczanie skryptu w części nagłówkowej dokumentu. Poniżej przykład:

<html> <head> <script> function wyswietlKomunikat(komunikat) { window.alert(komunikat); } </script> </head> <body> </body> </html>

Pliki ze skryptami można dołączać do dokumentów HTML przy użyciu znacznika <script> oraz atrybutu src tego znacznika, co umożliwia utrzymanie porządku w dokumencie HTML i oddzielenie treści strony od skryptów. Poniżej przykład:

<html> <head> <script src="sciezkaDoPlikuZeSkryptami"> </script> </head> <body> </body> </html>

Kolejny sposób na dołączenie skryptu do strony html to dynamiczne załadowanie pliku ze skryptami. Można to zrobić w poniższy sposób:

(function() { var scriptElementToLoad = document.createElement('script'); scriptElementToLoad.type = 'text/javascript'; scriptElementToLoad.async = true; scriptElementToLoad.src = 'http://localhost/skrypty.js'; var scriptElement = document.getElementsByTagName('script')[0]; scriptElement.parentNode.insertBefore(scriptElementToLoad, scriptElement); return "Wczytano plik javascript: " + scriptElementToLoad.src; })();
