Deklaracja typu dokumentu

Na początku każdego dokumentu HTML czyli pliku zapisanego z rozszerzeniem .html powinna znaleźć się deklaracja typu dokumentu. Zgodnie ze standardem HTML 5 deklaracja typu dokumentu wygląda następująco:

<!DOCTYPE html> <html> <head> </head> <body> </body> </html>

W starszych dokumentach HTML 4 deklaracja typu dokumentu wyglądała następująco:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

lub

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

lub

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
