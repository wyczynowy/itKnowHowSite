Arkusze stylów - CSS

CSS - Kaskadowe Arkusze Stylów, to nazwa mechanizmu umożliwiającego zarządzanie stylami na stronach internetowych. Jest kilka sposobów na to aby dekorować elementy znajdujące się na stronach internetowych. Pierwszy sposób jaki umożliwia twórcom stron html dekorowanie elementów to wpisywanie stylu każdorazowo jako wartość atrybutu style jakiegoś elementu, np.:

<button style="color:red; border: 1px solid red; border-radius: 5px;">Przycisk</button>

Powyższa konstrukcja umożliwia umieszczenie w dokumencie przycisku, którego styl wyglądu jest opisany bezpośrednio jako wartość atrybutu style, Przycisk ten będzie się prezentował tak na stronie internetowej:

Drugi sposób to deklaracja stylu w nagłówku dokumentu, czyli pomiędzy znacznikami <head></head> jak pokazano to poniżej:

<html>
 <head>
  <style>
   .przycisk {
    color: green;
    border: 1px solid green;
    border-radius: 5px;
   }
  </style>
 </head>
 <body>
  <button class="przycisk">Przycisk</button>
 </body>
</html>

W powyższym przykładzie pokazano możliwość deklaracji stylu w nagłówku dokumentu. Powyższa konstrukcja deklaruje klasę, która będzie się cechowała określonym stylem. Oznacza to, że każdy element dokumentu html, który będzie miał określony atrybut class o wartości przycisk, będzie posiadał styl, taki jak to określono w powyższym przykładzie. Deklaracja stylu wpisanego w dokument musi zawsze znajdować się w nagłówku dokumentu pomiędzy znacznikami <style></code>

Trzeci sposób dekorowania elementów w dokumentach html to deklaracja stylów w zewnętrznym dokumencie HTML. Dzięki takiemu zabiegowi separujemy wygląd strony od treści strony co jest bardzo porządanym efektem. Dzięki temu nasz dokument HTML będzie bardziej przejrzysty i będziemy w stanie szybciej zlokalizować poszukiwaną treść, bądź dekorację. Poniżej przedstawiono sposób dołączenia do dokumentu HTML zewnętrznego arkusza stylów :

<html>
 <head>
  <link rel="Stylesheet" type="text/css" href="style.css"/>
 </head>
 <body>
  <button id="przycisk">Przycisk</button>
 </body>
</html>

Zawartość pliku style.css znajdującego się w tym samym katalogu co plik dokumentu HTML

#przycisk {
 color: blue;
 border: 1px solid blue;
 border-radius: 5px;
}

Jak widać powyżej, użycie znacznika <link></link> umożliwia dołączenie zewnętrznego arkusza stylów CSS do dokumentu HTML. Atrybut rel informuje, że dołączamy arkusz stylów, atrybut type określa typ dołączanego arkusza a atrybut href określa lokalizację pliku względem dokumentu HTML, do którego jest dołączany. Wartość atrybutu href może zarówno wskazywać na lokalizację w internecie.
