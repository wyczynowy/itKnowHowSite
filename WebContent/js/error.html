Obiekt Error

Obiekty błędów tworzy się za pomoca konstruktorów: EvalError, RangeError, ReferenceError, SyntaxError, TypeError i URIError. Wszystkie konstruktory dziedziczą po Error. Obiekt Error możemy wykorzystać w następujący sposób:

try { funkcjaKtorejNieMa(); } catch (e) { // Obsluga bledu } finally { // Blok finally }

Mozna tworzyc swoje bledy w nastepujacy sposob: throw new Error('Nazwa nowego bledu ktora zostanie wyrzucona');
