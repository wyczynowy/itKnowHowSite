Date

Obiekt Date wspomaga wszelkiego rodzaju operacje związane z datą i czasem w języku javascript. Obiekt typu Date tworzy się standardowoo za pomocą operatora new:

var data = new Date()
data >> Date 2018-07-15T20:30:52.159Z

Obiekty typu Date można tworzyć na różne sposoby:

new Date('2009 11 12')
new Date('1 1 2012')
new Date('1 mar 2012')
new Date(rok, miesiac, dzien, godzina, minuta, sekunda, milisekunda) - miesiace liczone od 0
new Date(liczbaMilisekundOdPoczatkuEryUniksa)

Obiekt Date posiada oczywiście szereg metod ułatwiających pracę z nim, np.:

data.setMonth(arg) - ustawia miesiąc
data.getMonth(arg) - pobiera miesiąc
Date.parse('Jan 1, 2008') - zamienia tekst na znacznik czasu
Date.UTC(2008, 0, 1) - czas uniwersalny
