String

Zmienne typu String w języku javascript tworzymy przy użyciu następującej składni:

var string = "Imię"
string >> "Imię" string[0] >> "I" string[3] >> "ę"

Innym sposobem tworzenia zmiennych typu String jest użycie konstruktora obiektu String:

var string = new String("Imię")
string >> "Imię" string[0] >> "I" string[3] >> "ę"

Nieważne w jaki sposób tworzymy zmienną typu String jest ona zawsze obiektem. W związku z tym, że jest to obiekt istnieją predefiniowane funkcje, które można wywoływać na obiekcie typu String oraz specjalne pola:

text.length - zwraca dlugość łancucha znaków
text.toUpperCase() - zamiana na duze litery
text.toLowerCase() - zamiana na male litery
text.charAt(arg) - zwraca znak znajdujący się na określonej pozycji
text.indexOf('arg') - przeszukuje ciąg znaków w poszukiwaniu łańcucha znaków 'arg'
text.indexOf('arg', poz) - przeszukuje ciąg znaków w poszukiwaniu łańcucha znaków 'arg' zaczynajac od pozycji poz
text.lastIndexOf('arg') - przeszukuje ciąg znaków w poszukiwaniu łańcucha znaków zaczynając od końca łańcucha znaków
text.slice(arg0, arg1) - zwraca tekst znajdujący się na pozycji określonej pomiędzy arg0 oraz arg1. Można użyć w inny sposób podając argumenty np.(1, -1) co oznacza, że funkcja ma zwrócic ciag znakow od pozycji 1 do pozycji text.lenght - 1
text.substring(arg0, arg1) - j.w.
text.split(" ") - zwraca talice znakow.
text.join(" ") - zamienia tablice w obiekt typu String łącząc elementy za pomocą " "
text.concat('arg') - łączy ciąg znaków text z ciągiem znaków podanych jako argument funkcji
text.search(regexp) - zwraca wynik wyszukiwania. regexp jest wyrażeniem regularnym
text.match(regexp) - zwraca tablicę dopasowań. regexp jest wyrażeniem regularnym:

var text = "Każdy kot ma jakieś imię"
text.match(/a/) >> Array [ "a" ] text.match(/a/g) >> Array(3) [ "a", "a", "a" ]

text.repleace(regexp, arg) - zamienia część ciągu znaków na arg zgodnie z wyrażeniem regularnym

var text = "Każdy kot ma jakieś imię"
text.replace(/[A-Z]/, '') >> "każdy kot ma jakieś imię" text.replace(/[A-Z]/g, "_$&") >> "_Każdy_kot_ma_jakieś_imię"

text.replace(/[A-Z]/g, "_&") - Poprzedza dopasowany tekst podkreślnikiem dzięki sekwencji $&

var csv = 'raz, dwa, trzy ,cztery';
csv.split(/\s*,\s*/) >> ["jeden","dwa","trzy","cztery"]

csv.split(/\s*,\s*/) - usunięcie wszystkich spacji. Wyrażenie \s* oznacza zero lub więcej spacji
