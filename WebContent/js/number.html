Number

Number jest typem danych liczbowych w javascript. Zmienną typu number można utworzyć w następujący sposób:

var number = 1;

lub przy użyciu funkcji Number:

var number = Number(1);

Można również utworzyć nowy obiekt za pomocą operatora new:

var numberObiekt = new Number(1);

Z typu danych Number możemy wydobyć kilka ciekawych informacji za pomocą odwołania w następujący sposób:

Number.MAX_VALUE - zwraca maksymalną wartość jaką może przechowywać ten typ danych,
Number.MIN_VALUE - zwraca minimalną wartość jaką może przechowywać ten typ danych,
Number.NEGATIVE_INFINITY - zwraca ujemna nieskonczonosc,
Number.NaN - zwraca NaN,

Na obiekcie możemy wywołać kilka metod, które mogą się przydać podczas pracy z nim:

obiekt.toFixed(1) - zaokragla liczbę do ilości miejsc po przecinku podanych jako argument tej funkcji,
obiekt.toPrecision(1) - zwraca łańcuch znaków reprezentujący dany obiekt z dokładnością do ilości znaków określoną jako argument funkcji
obiekt.toExponential() - zamienia liczbę na liczbę w postaci wykładnikowej,
obiekt.toString(arg)- umozliwia zamianę na inną postać. Jako argument podać postać konwersji,
