Obiekt

Obiekty w języku javascript tworzymy przy użyciu następującej składni:

var obiekt = {};

Powyżej został utworzony pusty obiekt o nazwie obiekt. Można również utworzyć obiekt przy użyciu operatora new:

var obiekt = new Object();

lub tworząc nowy obiekt zadeklarowac jego pola i przypisać im określone wartości:

var samochod = { rodzaj: 'Osobowy', kolor: 'Czarny', prezentujSie: function() { return 'Samochod z kategorii ' + samochod.rodzaj + ', koloru '+ samochod.kolor; }

Można również tworzyć obiekty przy użyciu konstruktora:

function Samochod(rodzaj, kolor) { this.rodzaj = rodzaj; this.kolor = kolor; this.prezentujSie = function() { return 'Samochod z kategorii ' + this.rodzaj + ', koloru ' + this.kolor; } } var samochod = new Samochod('Osobowy', 'Czarny')

Kolejnym sposobem na tworzenie obiektów w języku javascript jest przygotowanie funkcji, która zwraca nam obiekt:

function factory(rodzaj, kolor) { return { rodzaj: rodzaj, kolor: kolor, przedstawSie: function() { return 'Samochod z kategorii ' + this.rodzaj + ', koloru ' + this.kolor; } }; }
