Operacje na dokumencie

Na dokumencie można wykonać sporo bardzo ciekawych operacji. Niektóre z nich poniżej:

document - obiekt dokumentu (wyświetla stronę jako obiekt DOM)
document.documentElement - wyświetla korzeń dokumentu czyli <html>
document.documentElement.nodeType - wyświetla typ węzła
document.documentElement.nodeName - wyświetla nazwę znacznika
document.documentElement.tagName - j.w.
document.documentElement.nodeValue - wyświetla wartość węzła
document.documentElement.hasChildNodes() - wyświetla informacje czy dany węzeł ma dziecko. W tym przypadku będzie wyświetlona informacja o dzieciach korzenia
document.documentElement.childNodes.length - wyświetla ilość dzieci danego węzła. W tym przypadku korzenia
document.documentElement.childNodes[arg] - dostęp do elementów dzieci danego węzla przy użyciu tablicopodobnej kolekcji
document.documentElement.childNodes[arg].parentNode - uzyskujemy dostęp do rodzica węzła childNodes[arg]
document.documentElement.childNodes[arg].childNodes.length - wyświetla informacje o ilości dzieci dziecka elementu korzeń czyli np. info o ilości wezłów w elemencie <body>
document.documentElement.childNodes[arg].hasAttributes() - wyświetla informacje czy węzeł posiada jakieś atrybuty
document.documentElement.childNodes[arg].attributes.length - wyświetla informacje o ilości atrybutów danego węzła
document.documentElement.childNodes[arg].attributes[arg1].nodeName - wyświetla nazwe atrybutu
document.documentElement.childNodes[arg].attributes[arg1].nodeValue - wyświetla wartość atrybutu
ocument.documentElement.childNodes[arg].getAttribute['nazwaAtrybutu'].nodeName/nodeValue - j.w.
document.documentElement.childNodes[arg].innerHTML/textContent - wyświetla text zawarty pomiędzy znacznikiem początkowym a końcowym węzla (textContent nie działa w IE)
document.getElementsByTagName('p') - zwraca tablice elementów o nazwie 'p' czyli w tym przypadku <p>
document.getElementsByTagName('p').length - zwraca ilość elementów tablicy
document.getElementsByTagName('p')[0] - zwraca cały węzeł
document.getElementsByTagName('p')[0].innerHTML - wyświetla zawartość elementu oraz umożliwia jego zmianę
document.getElementsByTagName('p')[0].className - zwraca wartość atrybutu class elementu <p>
document.getElementsByTagName('p')[0].id - zwraca wartość atrybutu id elementu <p>
document.all - umożliwia pobranie wszystkich elementów (dziala w starszych przeglądarkach IE)
document.getElementsByTagName('*') - j.w. dla przeglądarek od IE7
document.getElementById(arg) - zwraca element o id = arg
document.getElementById(arg).element.nextSibling/previousSlibing - umożliwia przejście do następnego/poprzedniego elementu rodzeństwa. Można wywoływać tak: element.nextlibling.nextSlibing.nextSlibing
document.body - zwraca element <body>
document.body.firstChild/lastChild - zwracaja odpowiednio pierwsze dziecko/ostatnie dziecko
document.getElementById(arg).style.border = '1px solid red' - zmienia styl obramowania elementu o id = arg
document.getElementById(arg).style.fontWeight = 'bold' - zmienia pogrubienie czcionki
document.getElementById(arg).align = 'right' - zmienia wyrównanie tekstu
document.getElementById(arg).name = 'nowaNazwa' - zmienia nazwę elementu
document.getElementById(arg).id - dostęp do id
document.createElement('t') - tworzy i zwraca element <t> document.body.appendChild(arg) - dodaje nowo utworzony element do <body> za ostatnim dotychczas dzieckiem
document.createTextNode() - tworzy nowy węzeł tekstowy
cloneNode(true/false) - kopiuje element. true - kopia z dziecmi / false - kopia plytka. Metode wywołuje się na elemencie do skopiowania
document.body.insertBefore(arg0, arg1) - dodaje element arg0 przed element arg1
document.body.removeChild(arg0) - usuwa element arg0 z <body>. Funkcja zwraca ten usunięty element
document.body.replaceChild(arg0, arg1) - metoda usuwa z <body> element arg0 i zamiast niego wkłada element arg1. Zwraca arg0
removeAll(document.body) - usuwa całą zawartość <body>
document.write() - pozwala na dodanie do strony kod w czasie jej ładowania. Metoda wywołana po załadowaniu strony nadpisze cała jej zawartość
document.writeln() - j.w. tylko dodaje znak nowej linii na końcu /n na koncu
document.cookie - wyświetla łańcuch znaków ciasteczek
document.title - zmienia tytuł strony
document.domain - pozwala odczytac domenę

Starsze sposoby dostępu do dokumentu

document.images - pobiera wszystkie obrazki ze strony -> odpowiada document.getElementsByTagName('img')
document.applets - odpowiada document.getElementsByTagName('applets')
document.links - zwraca wszystkie linki ze strony
document.anchors - zwraca wszystkie zakotwiczenia ze strony
document.forms - zwraca wszystkie znaczniki form ze strony
document.forms[0].elements[arg] - zwraca przyciski i pola tekstowe formularza
document.forms[0].elements['nazwaElementu'] - notacja tablicowa umożliwia odwołanie się do elementu po nazwie name , jeżeli element takową nazwę posiada
document.forms[0].elements.nazwaElementu - odwołanie się bezpośrednio do pola obiektu (j.w.)
